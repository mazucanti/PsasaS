O projeto conta com 6 módulos: Construcao.py, Implementacao.py, Projeto.py, Requisitos.py, Teste.py e RODE-ME.py
Cada módulo está associado a um subprocesso do processo de Software, exceto RODE-ME.py
Este módulo importa os demais, que estão na pasta Etapas, na ordem apropriada para o funcionamento do programa

Se a organização das pastas não for alterada, só se faz necessária a execução do RODE-ME.py
Caso a organização de pastas seja alterada, o código pode não encontrar os módulos e não funcionar corretamente
